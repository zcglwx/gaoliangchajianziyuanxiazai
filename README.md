# 高亮插件资源下载

## 资源介绍

本仓库提供了一个高亮插件资源文件的下载，包含以下两个插件：

1. **HighlightPlus (URP)**
2. **OutLine (HDRP)**

这两个插件分别适用于Unity的Universal Render Pipeline (URP) 和 High Definition Render Pipeline (HDRP)，能够帮助开发者轻松实现物体的高亮效果和轮廓线效果。

## 使用说明

1. **HighlightPlus (URP)**：
   - 适用于URP项目，能够为游戏中的物体添加高亮效果。
   - 支持多种高亮模式和自定义设置，方便开发者根据需求调整效果。

2. **OutLine (HDRP)**：
   - 适用于HDRP项目，能够为物体添加轮廓线效果。
   - 提供多种轮廓线样式和颜色选择，增强游戏的视觉效果。

## 安装方法

1. 下载本仓库中的资源文件。
2. 将下载的文件导入到你的Unity项目中。
3. 根据项目使用的渲染管线（URP或HDRP），选择相应的插件进行配置和使用。

## 注意事项

- 请确保你的Unity项目已经配置了相应的渲染管线（URP或HDRP），否则插件可能无法正常工作。
- 在使用插件时，建议参考插件自带的文档或示例场景，以便更好地理解和使用插件的功能。
